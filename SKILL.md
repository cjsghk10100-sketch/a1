# SKILL.md - Skill Hub

이 문서는 이 워크스페이스의 스킬 실행 허브입니다.

## SSOT 계층
1. `memory/ops/COMMON_CONSTITUTION_V1.md` (최상위 규범 SSOT)
2. `memory/ops/MEMORY_RUNTIME_SSOT.md` (운영 SSOT)
3. `memory/state.md` (현재 상태 SSOT)

## 1층: Core Runtime Skills
- 메모리 관리 (daily/monthly/weekly/state)
- 결정→실행→증거 기록 루프
- 실패/교훈 축적

## 2층: Domain Skills
- 업무/프로젝트별 운영 스킬
- 도메인별 체크리스트와 실행 순서
- 참고 문서: `references/`

## 3층: Automation Skills
- 반복 작업 자동화
- 스크립트 연계 (`scripts/`)
- 주기 작업/상태 점검

## Skill 작성 규칙
1. 입력/출력 형식을 명시한다.
2. 실패 조건과 롤백 방법을 적는다.
3. 실행 증거(로그/파일 경로)를 남긴다.
4. 템플릿은 `memory/ops/` 또는 `memory/reference/`에 둔다.

## 공통 운영 스킬
- 공통 스킬: `memory/ops/COMMON_SKILL_V1.md`
- 공통 헌법: `memory/ops/COMMON_CONSTITUTION_V1.md`
