# MEMORY_PROMOTION_RULES_V1

목적: daily의 중요한 내용을 장기 기억으로 체계적으로 승격한다.

## 태그 규칙
- daily 기록 시 아래 태그 사용:
  - [PROMOTE] 장기 기억 승격 후보
  - [KEEP-DAILY] 일회성, daily 유지
  - [VERIFY] 사실 검증 필요

## 승격 기준
1. 세션/주간을 넘어 반복 가치가 있는가
2. 의사결정 품질/실수 감소에 기여하는가
3. 향후 자동화/ROI 개선에 직접 연결되는가

## 주간 처리
- [PROMOTE] 항목을 주 1회 검토
- 검토 통과 시 `MEMORY.md`에 요약 승격
- 중복 내용은 기존 MEMORY 항목 업데이트로 통합
